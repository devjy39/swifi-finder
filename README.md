 # Seoul Wifi Finder
 
 - 서울 WIFI 정보를 가져와서 현재 위치(lan, lnt)에서 가까운 순으로 출력해준다.
 - 사용 openAPI : https://data.seoul.go.kr/dataList/OA-20883/S/1/datasetView.do <br/>
 - WIFI 조회 history 보기
  - history 삭제
 
 #### Setting
 - IDE : IntelliJ
 - DB : SQLite3
 - 순수 JAVA, JSP
 - WAS : tomcat8.5
 - Library : servlet-jsp, gson, lombok, sqlite-jdbc
